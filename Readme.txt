Android性能测试工具目前实现的功能包括：
	测试项：①monkey测试，②启动时间测试，③手工测试时的资源监控；
	资源监控：①cpu、内存监控，②流量监控（电量监控暂时未实现）；
	制定图表：根据监控数据生成二维折线图表；
	生成报告：根据测试结果、数据、图表生成html格式的报告；
后期如果有其他需求，可以完善更新。

使用说明：
	①测试项中的两项全选时，先执行monkey测试，然后执行启动时间测试；两项都不选时，实现的是手工操作时的资源监控功能；
	②资源监控选项勾选后就会在后台记录数据，测试 完成后自动生成曲线图；


本工具的运行需要事先配置环境，所需环境配置如下：
	①安装Android SDK；
	②安装JDK；
	③安装Python，版本2.7+

注意：工具代码需要执行的话，文件存放的路径中一定不要，不要，不要包含中文。

执行方式：运行run.bat文件即可。

若有不尽不详之处，欢迎与自动化小组成员当面沟通。

谢谢使用！
